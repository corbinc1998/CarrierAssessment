# Assessment Brief

## Brief

- You have received an image containing two tables and have been instructed to place it on a website. However, simply placing the image on a website will not allow users to easily view the content.
- Your task is to create a **mobile-responsive** version of these tables, keeping in mind that most users will access the website from mobile devices.
- The **desktop version** must also be useful and accessible.
- There are no restrictions on how you approach the solution, so feel free to be creative. **Creativity is strongly encouraged!** Make sure your solution works well across different devices, operating systems, and browsers.

## Constraints

- Include a **README** explaining your approach.
- Prevent empty or duplicate tasks.

## Deliverables

A GitHub repository containing:

- Source code
- README
- Notes describing any trade-offs and/or what you would add with more time
- A direct link to the solution webpage
